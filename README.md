# 🏠 House Price Prediction 

## 📌 Overview
This project focuses on predicting house prices using Machine Learning techniques. It covers the complete data science pipeline including data cleaning, feature engineering, exploratory data analysis (EDA), and model building.

---

## 📊 Dataset
- 21,000+ records with 19+ features  
- Includes attributes like location, number of rooms, property age, renovation status, etc.  
- Data preprocessing includes handling missing values and type conversions  

---

## ⚙️ Key Features
- Data Cleaning & Preprocessing  
- Feature Engineering (Property Age, Renovation Status)  
- Exploratory Data Analysis (EDA)  
- Model Building & Evaluation  
- Geospatial Analysis  

---

## 🧠 Models Used
- Linear Regression  
- Ridge Regression  
- Random Forest Regressor  

All models were implemented using **Scikit-learn pipelines** with scaling and transformations.

---

## 📈 Performance
- **R² Score:** 0.88  
- Low MAE & RMSE ensuring reliable predictions  

---

## 📉 Exploratory Data Analysis
- Used Seaborn heatmaps to identify correlations  
- Applied Folium maps for geospatial clustering  
- Identified strong impact of location on pricing  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Seaborn, Matplotlib  
- Folium

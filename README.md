# 📊 Customer Churn Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting customer churn (whether a customer will leave a service) using Machine Learning techniques. It covers the complete data science pipeline including data cleaning, feature engineering, exploratory data analysis (EDA), and model building.

---

## 📊 Dataset
- Banking dataset with customer demographic and account information  
- Includes features like credit score, geography, gender, age, tenure, balance, and activity status  
- Data preprocessing includes handling categorical variables, encoding, and feature scaling  

---

## ⚙️ Key Features
- Data Cleaning & Preprocessing  
- Feature Engineering & Encoding  
- Exploratory Data Analysis (EDA)  
- Model Building & Evaluation  
- Real-time Prediction using Streamlit  

---

## 🧠 Models Used
- Decision Tree Classifier  
- Random Forest Classifier  

Models were implemented using **Scikit-learn**, with proper preprocessing and evaluation techniques.

---

## 📈 Performance
- Achieved strong classification performance using Random Forest  
- Evaluated using Accuracy, Precision, Recall, and ROC-AUC  
- Used probability-based prediction (`predict_proba`) for better decision-making  

---

## 📉 Exploratory Data Analysis
- Identified key churn factors such as inactivity and low product usage  
- Observed higher churn rates in specific geographic regions  
- Analyzed feature relationships using correlation and visualizations  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Streamlit

